# Advanced Web Applications Development

Java web applications developed at Florida State University using enterprise web development technologies.

## Technologies

- Java
- JSP
- Java Servlets
- Apache Tomcat
- JDBC
- MySQL
- HTML/CSS/JavaScript
- MVC Architecture

## Key Features

- Database-driven web applications
- CRUD operations
- Client-side validation
- Server-side validation
- Prepared Statements
- SQL Injection Prevention
- XSS Protection

## Projects

### Customer Management System
Built a Java web application connected to a MySQL database using JDBC.

Features:
- Create customer records
- Modify existing records
- Database validation
- MVC architecture

### Security Implementation
Implemented:
- Prepared statements
- JSTL escaping
- Input validation
